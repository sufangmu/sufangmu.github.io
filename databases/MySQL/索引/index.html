    зҙўеј•

    дёҖгҖҒжҰӮиҝ°¶

    MySQLе®ҳж–№еҜ№зҙўеј•зҡ„е®ҡд№үдёәпјҡзҙўеј•пјҲindexпјүжҳҜеё®еҠ©MySQLй«ҳж•ҲиҺ·еҸ–ж•°жҚ®зҡ„ж•°жҚ®з»“жһ„пјҲжңүеәҸпјүгҖӮеңЁж•°жҚ®д№ӢеӨ–пјҢж•°жҚ®еә“зі»з»ҹиҝҳз»ҙжҠӨиҖ…ж»Ўи¶ізү№е®ҡжҹҘжүҫз®—жі•зҡ„ж•°жҚ®з»“жһ„пјҢиҝҷдәӣж•°жҚ®з»“жһ„д»Ҙжҹҗз§Қж–№ејҸеј•з”ЁпјҲжҢҮеҗ‘пјүж•°жҚ®пјҢ иҝҷж ·е°ұеҸҜд»ҘеңЁиҝҷдәӣж•°жҚ®з»“жһ„дёҠе®һзҺ°й«ҳзә§жҹҘжүҫз®—жі•пјҢиҝҷз§Қж•°жҚ®з»“жһ„е°ұжҳҜзҙўеј•гҖӮеҰӮдёӢйқўзҡ„==зӨәж„Ҹеӣҫ==жүҖзӨә :

    е·Ұиҫ№жҳҜж•°жҚ®иЎЁпјҢдёҖе…ұжңүдёӨеҲ—дёғжқЎи®°еҪ•пјҢжңҖе·Ұиҫ№зҡ„жҳҜж•°жҚ®и®°еҪ•зҡ„зү©зҗҶең°еқҖпјҲжіЁж„ҸйҖ»иҫ‘дёҠзӣёйӮ»зҡ„и®°еҪ•еңЁзЈҒзӣҳдёҠд№ҹе№¶дёҚжҳҜдёҖе®ҡзү©зҗҶзӣёйӮ»зҡ„пјүгҖӮдёәдәҶеҠ еҝ«Col2зҡ„жҹҘжүҫпјҢеҸҜд»Ҙз»ҙжҠӨдёҖдёӘеҸіиҫ№жүҖзӨәзҡ„дәҢеҸүжҹҘжүҫж ‘пјҢжҜҸдёӘиҠӮзӮ№еҲҶеҲ«еҢ…еҗ«зҙўеј•й”®еҖје’ҢдёҖдёӘжҢҮеҗ‘еҜ№еә”ж•°жҚ®и®°еҪ•зү©зҗҶең°еқҖзҡ„жҢҮй’ҲпјҢиҝҷж ·е°ұеҸҜд»Ҙиҝҗз”ЁдәҢеҸүжҹҘжүҫеҝ«йҖҹиҺ·еҸ–еҲ°зӣёеә”ж•°жҚ®гҖӮ

    дёҖиҲ¬жқҘиҜҙзҙўеј•жң¬иә«д№ҹеҫҲеӨ§пјҢдёҚеҸҜиғҪе…ЁйғЁеӯҳеӮЁеңЁеҶ…еӯҳдёӯпјҢеӣ жӯӨзҙўеј•еҫҖеҫҖд»Ҙзҙўеј•ж–Үд»¶зҡ„еҪўејҸеӯҳеӮЁеңЁзЈҒзӣҳдёҠгҖӮзҙўеј•жҳҜж•°жҚ®еә“дёӯз”ЁжқҘжҸҗй«ҳжҖ§иғҪзҡ„жңҖеёёз”Ёзҡ„е·Ҙе…·гҖӮ

    дәҢгҖҒзҙўеј•дҪңз”Ё¶

    1. дјҳеҠҝ¶

    1. зұ»дјјдәҺд№ҰзұҚзҡ„зӣ®еҪ•зҙўеј•пјҢжҸҗй«ҳж•°жҚ®жЈҖзҙўзҡ„ж•ҲзҺҮпјҢйҷҚдҪҺж•°жҚ®еә“зҡ„IOжҲҗжң¬гҖӮ
    2. йҖҡиҝҮзҙўеј•еҲ—еҜ№ж•°жҚ®иҝӣиЎҢжҺ’еәҸпјҢйҷҚдҪҺж•°жҚ®жҺ’еәҸзҡ„жҲҗжң¬пјҢйҷҚдҪҺCPUзҡ„ж¶ҲиҖ—гҖӮ

    2. еҠЈеҠҝ¶

    1. е®һйҷ…дёҠзҙўеј•д№ҹжҳҜдёҖеј иЎЁпјҢиҜҘиЎЁдёӯдҝқеӯҳдәҶдё»й”®дёҺзҙўеј•еӯ—ж®өпјҢе№¶жҢҮеҗ‘е®һдҪ“зұ»зҡ„и®°еҪ•пјҢжүҖд»Ҙзҙўеј•еҲ—д№ҹжҳҜиҰҒеҚ з”Ёз©әй—ҙзҡ„гҖӮ

    2. иҷҪз„¶зҙўеј•еӨ§еӨ§жҸҗй«ҳдәҶжҹҘиҜўж•ҲзҺҮпјҢеҗҢж—¶еҚҙд№ҹйҷҚдҪҺжӣҙж–°иЎЁзҡ„йҖҹеәҰпјҢеҰӮеҜ№иЎЁиҝӣиЎҢINSERTгҖҒUPDATEгҖҒDELETEгҖӮеӣ дёәжӣҙж–°иЎЁж—¶пјҢMySQL дёҚд»…иҰҒдҝқеӯҳж•°жҚ®пјҢиҝҳиҰҒдҝқеӯҳдёҖдёӢзҙўеј•ж–Үд»¶жҜҸж¬Ўжӣҙж–°ж·»еҠ дәҶзҙўеј•еҲ—зҡ„еӯ—ж®өпјҢйғҪдјҡи°ғж•ҙеӣ дёәжӣҙж–°жүҖеёҰжқҘзҡ„й”®еҖјеҸҳеҢ–еҗҺзҡ„зҙўеј•дҝЎжҒҜгҖӮ

    дёүгҖҒзҙўеј•з»“жһ„¶

    зҙўеј•жҳҜеңЁMySQLзҡ„еӯҳеӮЁеј•ж“ҺеұӮдёӯе®һзҺ°зҡ„пјҢиҖҢдёҚжҳҜеңЁжңҚеҠЎеҷЁеұӮе®һзҺ°зҡ„гҖӮжүҖд»ҘжҜҸз§ҚеӯҳеӮЁеј•ж“Һзҡ„зҙўеј•йғҪдёҚдёҖе®ҡе®Ңе…ЁзӣёеҗҢпјҢд№ҹдёҚжҳҜжүҖжңүзҡ„еӯҳеӮЁеј•ж“ҺйғҪж”ҜжҢҒжүҖжңүзҡ„зҙўеј•зұ»еһӢзҡ„гҖӮMySQLзӣ®еүҚжҸҗдҫӣдәҶд»ҘдёӢ4з§Қзҙўеј•пјҡ

    • BTREE зҙўеј• пјҡ жңҖеёёи§Ғзҡ„зҙўеј•зұ»еһӢпјҢеӨ§йғЁеҲҶзҙўеј•йғҪж”ҜжҢҒ B ж ‘зҙўеј•гҖӮ

    • HASH зҙўеј•пјҡеҸӘжңүMemoryеј•ж“Һж”ҜжҢҒ пјҢ дҪҝз”ЁеңәжҷҜз®ҖеҚ• гҖӮ

    • R-tree зҙўеј•пјҲз©әй—ҙзҙўеј•пјүпјҡз©әй—ҙзҙўеј•жҳҜMyISAMеј•ж“Һзҡ„дёҖдёӘзү№ж®Ҡзҙўеј•зұ»еһӢпјҢдё»иҰҒз”ЁдәҺең°зҗҶз©әй—ҙж•°жҚ®зұ»еһӢпјҢйҖҡеёёдҪҝз”Ёиҫғе°‘пјҢдёҚеҒҡзү№еҲ«д»Ӣз»ҚгҖӮ
    • Full-text пјҲе…Ёж–Үзҙўеј•пјү пјҡе…Ёж–Үзҙўеј•д№ҹжҳҜMyISAMзҡ„дёҖдёӘзү№ж®Ҡзҙўеј•зұ»еһӢпјҢдё»иҰҒз”ЁдәҺе…Ёж–Үзҙўеј•пјҢInnoDBд»ҺMysql5.6зүҲжң¬ејҖе§Ӣж”ҜжҢҒе…Ёж–Үзҙўеј•гҖӮ

    Full-text 5.6зүҲжң¬д№ӢеҗҺж”ҜжҢҒ ж”ҜжҢҒ дёҚж”ҜжҢҒ

    жҲ‘д»¬е№іеёёжүҖиҜҙзҡ„зҙўеј•пјҢеҰӮжһңжІЎжңүзү№еҲ«жҢҮжҳҺпјҢйғҪжҳҜжҢҮB+ж ‘пјҲеӨҡи·Ҝжҗңзҙўж ‘пјҢе№¶дёҚдёҖе®ҡжҳҜдәҢеҸүзҡ„пјүз»“жһ„з»„з»Үзҡ„зҙўеј•гҖӮе…¶дёӯиҒҡйӣҶзҙўеј•гҖҒеӨҚеҗҲзҙўеј•гҖҒеүҚзјҖзҙўеј•гҖҒе”ҜдёҖзҙўеј•й»ҳи®ӨйғҪжҳҜдҪҝз”Ё B+tree зҙўеј•пјҢз»ҹз§°дёә зҙўеј•гҖӮ

    1. BTREE з»“жһ„¶

    BTreeеҸҲеҸ«еӨҡи·Ҝе№іиЎЎжҗңзҙўж ‘пјҢдёҖйў—mеҸүзҡ„BTreeзү№жҖ§еҰӮдёӢпјҡ

    • ж ‘дёӯжҜҸдёӘиҠӮзӮ№жңҖеӨҡеҢ…еҗ«mдёӘеӯ©еӯҗгҖӮ
    • йҷӨж №иҠӮзӮ№дёҺеҸ¶еӯҗиҠӮзӮ№еӨ–пјҢжҜҸдёӘиҠӮзӮ№иҮіе°‘жңү[ceil(m/2)]дёӘеӯ©еӯҗгҖӮ
    • иӢҘж №иҠӮзӮ№дёҚжҳҜеҸ¶еӯҗиҠӮзӮ№пјҢеҲҷиҮіе°‘жңүдёӨдёӘеӯ©еӯҗгҖӮ
    • жүҖжңүзҡ„еҸ¶еӯҗиҠӮзӮ№йғҪеңЁеҗҢдёҖеұӮгҖӮ
    • жҜҸдёӘйқһеҸ¶еӯҗиҠӮзӮ№з”ұnдёӘkeyдёҺn+1дёӘжҢҮй’Ҳз»„жҲҗпјҢе…¶дёӯ[ceil(m/2)-1] <= n <= m-1

    д»Ҙ5еҸүBTreeдёәдҫӢпјҢkeyзҡ„ж•°йҮҸпјҡе…¬ејҸжҺЁеҜј[ceil(m/2)-1] <= n <= m-1гҖӮжүҖд»Ҙ 2 <= n <=4 гҖӮеҪ“n>4ж—¶пјҢдёӯй—ҙиҠӮзӮ№еҲҶиЈӮеҲ°зҲ¶иҠӮзӮ№пјҢдёӨиҫ№иҠӮзӮ№еҲҶиЈӮгҖӮ

    жҸ’е…Ҙ C N G A H E K Q M F W L T Z D P R X Y S ж•°жҚ®дёәдҫӢгҖӮ

    жј”еҸҳиҝҮзЁӢеҰӮдёӢпјҡ

    1). жҸ’е…ҘеүҚ4дёӘеӯ—жҜҚ C N G A

    2). жҸ’е…ҘHпјҢn>4пјҢдёӯй—ҙе…ғзҙ Gеӯ—жҜҚеҗ‘дёҠеҲҶиЈӮеҲ°ж–°зҡ„иҠӮзӮ№

    3). жҸ’е…ҘEпјҢKпјҢQдёҚйңҖиҰҒеҲҶиЈӮ

    4). жҸ’е…ҘMпјҢдёӯй—ҙе…ғзҙ Mеӯ—жҜҚеҗ‘дёҠеҲҶиЈӮеҲ°зҲ¶иҠӮзӮ№G

    5). жҸ’е…ҘFпјҢWпјҢLпјҢTдёҚйңҖиҰҒеҲҶиЈӮ

    6). жҸ’е…ҘZпјҢдёӯй—ҙе…ғзҙ Tеҗ‘дёҠеҲҶиЈӮеҲ°зҲ¶иҠӮзӮ№дёӯ

    7). жҸ’е…ҘDпјҢдёӯй—ҙе…ғзҙ Dеҗ‘дёҠеҲҶиЈӮеҲ°зҲ¶иҠӮзӮ№дёӯгҖӮз„¶еҗҺжҸ’е…ҘPпјҢRпјҢXпјҢYдёҚйңҖиҰҒеҲҶиЈӮ

    8). жңҖеҗҺжҸ’е…ҘSпјҢNPQRиҠӮзӮ№n>5пјҢдёӯй—ҙиҠӮзӮ№Qеҗ‘дёҠеҲҶиЈӮпјҢдҪҶеҲҶиЈӮеҗҺзҲ¶иҠӮзӮ№DGMTзҡ„n>5пјҢдёӯй—ҙиҠӮзӮ№Mеҗ‘дёҠеҲҶиЈӮ

    еҲ°жӯӨпјҢиҜҘBTREEж ‘е°ұе·Із»Ҹжһ„е»әе®ҢжҲҗдәҶпјҢ BTREEж ‘ е’Ң дәҢеҸүж ‘ зӣёжҜ”пјҢ жҹҘиҜўж•°жҚ®зҡ„ж•ҲзҺҮжӣҙй«ҳпјҢ еӣ дёәеҜ№дәҺзӣёеҗҢзҡ„ж•°жҚ®йҮҸжқҘиҜҙпјҢBTREEзҡ„еұӮзә§з»“жһ„жҜ”дәҢеҸүж ‘е°ҸпјҢеӣ жӯӨжҗңзҙўйҖҹеәҰеҝ«гҖӮ

    2. B+TREE з»“жһ„¶

    B+TreeдёәBTreeзҡ„еҸҳз§ҚпјҢB+TreeдёҺBTreeзҡ„еҢәеҲ«дёәпјҡ

    1). nеҸүB+TreeжңҖеӨҡеҗ«жңүnдёӘkeyпјҢиҖҢBTreeжңҖеӨҡеҗ«жңүn-1дёӘkeyгҖӮ

    2). B+Treeзҡ„еҸ¶еӯҗиҠӮзӮ№дҝқеӯҳжүҖжңүзҡ„keyдҝЎжҒҜпјҢдҫқkeyеӨ§е°ҸйЎәеәҸжҺ’еҲ—гҖӮ

    3). жүҖжңүзҡ„йқһеҸ¶еӯҗиҠӮзӮ№йғҪеҸҜд»ҘзңӢдҪңжҳҜkeyзҡ„зҙўеј•йғЁеҲҶгҖӮ

    з”ұдәҺB+TreeеҸӘжңүеҸ¶еӯҗиҠӮзӮ№дҝқеӯҳkeyдҝЎжҒҜпјҢжҹҘиҜўд»»дҪ•keyйғҪиҰҒд»Һrootиө°еҲ°еҸ¶еӯҗгҖӮжүҖд»ҘB+Treeзҡ„жҹҘиҜўж•ҲзҺҮжӣҙеҠ зЁіе®ҡгҖӮ

    3. MySQLдёӯзҡ„B+Tree¶

    • зҙўеј•еӯ—ж®өзҡ„йҖүжӢ©пјҢжңҖдҪіеҖҷйҖүеҲ—еә”еҪ“д»ҺwhereеӯҗеҸҘзҡ„жқЎд»¶дёӯжҸҗеҸ–пјҢеҰӮжһңwhereеӯҗеҸҘдёӯзҡ„з»„еҗҲжҜ”иҫғеӨҡпјҢйӮЈд№Ҳеә”еҪ“жҢ‘йҖүжңҖеёёз”ЁгҖҒиҝҮж»Өж•ҲжһңжңҖеҘҪзҡ„еҲ—зҡ„з»„еҗҲгҖӮ

    • дҪҝз”Ёе”ҜдёҖзҙўеј•пјҢеҢәеҲҶеәҰи¶Ҡй«ҳпјҢдҪҝз”Ёзҙўеј•зҡ„ж•ҲзҺҮи¶Ҡй«ҳгҖӮ

    • зҙўеј•еҸҜд»Ҙжңүж•Ҳзҡ„жҸҗеҚҮжҹҘиҜўж•°жҚ®зҡ„ж•ҲзҺҮпјҢдҪҶзҙўеј•ж•°йҮҸдёҚжҳҜеӨҡеӨҡзӣҠе–„пјҢзҙўеј•и¶ҠеӨҡпјҢз»ҙжҠӨзҙўеј•зҡ„д»Јд»·иҮӘз„¶д№ҹе°ұж°ҙж¶ЁиҲ№й«ҳгҖӮеҜ№дәҺжҸ’е…ҘгҖҒжӣҙж–°гҖҒеҲ йҷӨзӯүDMLж“ҚдҪңжҜ”иҫғйў‘з№Ғзҡ„иЎЁжқҘиҜҙпјҢзҙўеј•иҝҮеӨҡпјҢдјҡеј•е…ҘзӣёеҪ“й«ҳзҡ„з»ҙжҠӨд»Јд»·пјҢйҷҚдҪҺDMLж“ҚдҪңзҡ„ж•ҲзҺҮпјҢеўһеҠ зӣёеә”ж“ҚдҪңзҡ„ж—¶й—ҙж¶ҲиҖ—гҖӮеҸҰеӨ–зҙўеј•иҝҮеӨҡзҡ„иҜқпјҢMySQLд№ҹдјҡзҠҜйҖүжӢ©еӣ°йҡҫз—…пјҢиҷҪз„¶жңҖз»Ҳд»Қз„¶дјҡжүҫеҲ°дёҖдёӘеҸҜз”Ёзҡ„зҙўеј•пјҢдҪҶж— з–‘жҸҗй«ҳдәҶйҖүжӢ©зҡ„д»Јд»·гҖӮ

    • дҪҝз”Ёзҹӯзҙўеј•пјҢзҙўеј•еҲӣе»әд№ӢеҗҺд№ҹжҳҜдҪҝз”ЁзЎ¬зӣҳжқҘеӯҳеӮЁзҡ„пјҢеӣ жӯӨжҸҗеҚҮзҙўеј•и®ҝй—®зҡ„I/Oж•ҲзҺҮпјҢд№ҹеҸҜд»ҘжҸҗеҚҮжҖ»дҪ“зҡ„и®ҝй—®ж•ҲзҺҮгҖӮеҒҮеҰӮжһ„жҲҗзҙўеј•зҡ„еӯ—ж®өжҖ»й•ҝеәҰжҜ”иҫғзҹӯпјҢйӮЈд№ҲеңЁз»ҷе®ҡеӨ§е°Ҹзҡ„еӯҳеӮЁеқ—еҶ…еҸҜд»ҘеӯҳеӮЁжӣҙеӨҡзҡ„зҙўеј•еҖјпјҢзӣёеә”зҡ„еҸҜд»Ҙжңүж•Ҳзҡ„жҸҗеҚҮMySQLи®ҝй—®зҙўеј•зҡ„I/Oж•ҲзҺҮгҖӮ

    • еҲ©з”ЁжңҖе·ҰеүҚзјҖпјҢNдёӘеҲ—з»„еҗҲиҖҢжҲҗзҡ„з»„еҗҲзҙўеј•пјҢйӮЈд№ҲзӣёеҪ“дәҺжҳҜеҲӣе»әдәҶNдёӘзҙўеј•пјҢеҰӮжһңжҹҘиҜўж—¶whereеӯҗеҸҘдёӯдҪҝз”ЁдәҶз»„жҲҗиҜҘзҙўеј•зҡ„еүҚеҮ дёӘеӯ—ж®өпјҢйӮЈд№ҲиҝҷжқЎжҹҘиҜўSQLеҸҜд»ҘеҲ©з”Ёз»„еҗҲзҙўеј•жқҘжҸҗеҚҮжҹҘиҜўж•ҲзҺҮгҖӮ
