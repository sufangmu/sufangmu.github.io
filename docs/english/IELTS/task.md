#### 习惯⽤法造句

a so that b…
1. I get up early so that I can catch the bus.
2. He work**s** hard so that he can make more money.
3. He studies English hard so that he can pass IELTS.
4. I bought a new computer so that I ~~can~~(could) improve work ~~effect.~~(efficiency)

I wish I knew that …

1. I wish I knew how ~~work it is~~ it works.
2. I wish I knew how hard it is.

主 figure out sth.

1. I need to figure out this math problem before tomorrow.
2. She finally figured out why the program crashed.
3. Can you figure out the meaning of this sentence?
4. The engineer is trying to figure out a solution to the database issue.

if ⼀般现在时，⼀般将来时

1. If it rains tomorrow, I will get up early.
2. If you practice English every day, your English will improve soon.
